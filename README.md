Light reimplementation of <https://github.com/mgharbi/demosaicnet>.
Runs on python3 (tested on Anaconda for Ubuntu 14.03).
Other dependencies are listed in requirements

1. Install

```
make setup
```

2. Download the data

```
make download_data
```

3. Train a net

```
make train_bayer
```

4. Test a net

```
make test_bayer
```
